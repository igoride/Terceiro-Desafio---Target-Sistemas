# calculo-juros-atraso-csharp

Programa console para calcular juros sobre um valor vencido, considerando a quantidade de dias em atraso e aplicando uma multa fixa de 2,5% ao dia. O usuário informa a data de vencimento e o valor original.

Funcionalidades:

Solicitação da data de vencimento e valor.
Cálculo automático dos dias de atraso.
Aplicação da multa diária de 2,5% (juros simples).
Exibição detalhada do cálculo realizado.
Menu para permitir múltiplos cálculos.
